# Rust Cheat Base

A clean and minimal **Rust cheat base** designed for learning and building upon.
This project provides a and modular structure.

---

## Features
- Organized project structure
- Entity list parsing
- Basic ESP framework
- ImGui menu integration

---

## Requirements
- Windows 10/11
- Visual Studio 2022 (C++20 toolset)
- DirectX SDK
- Kernel Mode driver
